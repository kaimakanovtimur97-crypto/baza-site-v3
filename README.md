# БАЗА — многостраничный сайт

Статический сайт по макетам Figma. Чистый HTML/CSS/JS + PHP-обработчик формы. Готов к загрузке на Timeweb.

## Страницы

| Файл | Страница |
|---|---|
| index.html | Главная |
| uslugi.html | Услуги (каталог) |
| uslugi-ii.html | Услуга: обучение сотрудников ИИ |
| uslugi-reglamenty.html | Услуга: регламенты и инструкции |
| uslugi-modelirovanie.html | Услуга: моделирование бизнес-процессов |
| o-nas.html | О нас |
| otzyvy.html | Отзывы |
| keisy.html | Кейсы (с фильтром по категориям) |
| kontakty.html | Контакты (форма + способы связи) |

Все страницы перелинкованы: меню, футер, кнопки «Подробнее»/«Заказать», CTA ведут на нужные разделы.

## Формы (send.php)

Формы на всех страницах шлют заявку в `send.php` → письмо через PHP `mail()`.
Поля: имя, телефон (с маской), «суть проблемы / ситуация компании» (квалификация), скрытые: страница-источник и ханипот от ботов.

Настройка: в `send.php` поменяйте `$to` на свою почту. Если письма не доходят — включите почту для домена в панели Timeweb или подключите SMTP.

## Анимации

- плавное появление секций и карточек при скролле (IntersectionObserver, с каскадом);
- «дыхание» синих градиентных блобов и парящие плашки;
- живые аккордеоны (FAQ, отрасли, этапы работы на детальных страницах);
- ховеры карточек и «блик» на кнопках;
- слайдеры каруселей и фильтр кейсов с перестроением сетки;
- уважает prefers-reduced-motion.

## Публикация на Timeweb

Загрузите содержимое папки в `public_html` (файловый менеджер или FTP). PHP уже есть на любом тарифе — форма заработает сразу.

## Что можно добавить

- **Шрифт Gilroy**: файлы в `assets/fonts/` (Gilroy-Regular.woff2, Gilroy-Semibold.woff2) — подключение прописано, пока используется Manrope.
- **Фото основателя**: `assets/img/nikita.jpg` (539×280+) — на странице «О нас»; без файла показывается фирменная градиентная подложка.
- Ссылку на мессенджер Max в футере (сейчас `#`).
- Тексты аккордеонов и «Результатов» на детальных страницах написаны мной по смыслу — проверьте формулировки.
